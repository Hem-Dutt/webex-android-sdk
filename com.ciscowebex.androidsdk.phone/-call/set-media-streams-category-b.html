setMediaStreamsCategoryB

abstract fun setMediaStreamsCategoryB(numStreams: Int, quality: MediaStreamQuality): Unit

Set all Category-B streams to the specified params. If the number of existing B streams is less than numStreams, it will add B streams as necessary. If the number of existing B streams is more than numStreams, it will remove the extra B streams.

Parameters

numStreams - number of Category-B streams desired; all of them will have the same resolution.

quality - max resolution for this stream - MediaStreamQuality.

Since
3.5.0
